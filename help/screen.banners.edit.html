Banner Manager : New / Edit

--- Draft 453

This page let's create or edit a banner's details.

Toolbar Icons

Upload: Opens a new window which lets you upload an image from your computer to the images/banners folder.

Save: Saves the content and returns to Manager.

Apply: Saves the content and remains on the editing screen.

Close: Cancels the action. Any modification is lost. If opened to edit, the banner remains checked-in.


Details

Banner Name: Type here the name for the Banner.

Client Name: Select from the list of configured clients.
 
Impressions Purchased: Enter the number of times to display to banner into the input field, or tick the "Unlimited" checkbox for unlimited display.
 
Banner URL: Select the banner image to be displayed from the list of available banners. Images need to be stored in the "images/banners" folder to be available for selection. The selected image will be displayed at the bottom of the screen.
 
Show Banner: Select Yes or No.
 
Click URL: Enter here the full URL to the page which should open when the banner is clicked on.
 
Custom banner code:  You may enter here some code (See Developper Manual for more information).
 
Clicks: "Reset Clicks" button: clicking this button will reset the click count shown on the right for the banner to 0.

Banner Image: Image selected under "Banner URL" is displayed here for your convenience.

Banner Alt text: Type here the image alt text for the Banner.
