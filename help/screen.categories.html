Category Manager

-- Draft version

This page shows show a list of the Categories defined on your site. They are ordered by the section they belong to. The Select Section drop-down lets you display the categories belonging to one specific section at a time.

You may filter the categories by the section they belong to.

See this Help Screen for specific help for News Feeds and Contacts Category Manager.
Help Screen: News Feeds and Contacts Category Manager.

Toolbar:
Click the icons after checking the box next to a specific Category.

Publish and Unpublish : Same function as "Published" in the headings. You may check multiple checkboxes.

New : Create a new Category.

Edit : Edit an existing category parameters.

Move : Opens a new "Move Category" window where you can choose to move the specific Category and *all its Content Items* to another existing Section. You may click multiple checkboxes here.

Copy : Opens a new "Copy Category" window where you can choose to copy the specific Category and *all its Content Items* to another existing Section. You may click multiple checkboxes here. (The content items copied will keep the same name)

Delete: Lets you delete the category chosen. An Alert will pop to let you confirm this choice. This action can't be performed if the category contains Content Items. In this last case, you would have to delete the Content Items first.


Columns:
Category Name: This is the name of a category. Click the name to edit the category parameters. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit ,

Published: Whether a category is published or not. Click the icon to change the state.

Order: Order of the categories on the list (this has no effect on site display). You may change it by editing 2 # conjointly and clicking on the "Save Order" icon in the heading.

Access: This is the User access level. Public, Registered or Special.

Section: The name of the Section de category belongs to. Clicking on this name lets you edit the category's section's parameters.

Category ID: There is the ID for each category as defined in the database.

# Active: This is the number of published content items in the category.

#Trash: Number of Content Items belonging to this category and currently in the Trash.
