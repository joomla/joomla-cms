This page displays information about a client when you create or edit it.

Toolbar Icons:
Save: Click here to save your new or edited client's details.

Close: Cancels the action. Any modification is lost. If opened to edit, the client remains checked-in.


Details:
Client Name : Type here the name of the business which buys banner display from your site.

Contact Name: Type here the name of whom to contact about matters related to this client.

Contact Email: The e-mail of the person to contact. Will only be visible here.

Extra Info: Any other information about the client you wish to store. Only visible here.
