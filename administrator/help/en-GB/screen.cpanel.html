Control Panel

The Control Panel is the first page you see when you log into the Administrators area. At anytime, you can return to it by clicking the "Home" menu item in the top menu bar.  From the Control Panel are 9 shortcut icons to easy access the frequently used parts of the Administrator site.


Control Panel Icons:

Menu Manager - Click this icon to access the Menu Manager.  You will be able  to  edit any of your menus. 

Front Page Manager -  Click this icon to access the Front Page Manager.  You will be able to edit which Content items appear on your Frontpage.

Static Content Manager - Click this icon to access the Static Content Manager.  You will be able to edit these type of  items.

Section Manager - Click this icon to access the Section Manager.  You will be able to edit your Sections.

Category Manager -Click this icon to access the Category Manager. You will be able to edit your Categories.

All Content Items - Click this icon to access the All Content Items.  All Content Items  lists all of your Content Items.  You will be able to edit any of your Content Items.

Trash Manager - Click this icon to access the Trash manager.  You will be able to delete permanently or to restore any items in the Trash Manager.

Global Configuration - Click this icon to access the Global configuration.  You will be able to edit any of your Web site configurations.

Help - Click this icon to view the Index of any of the Help Screens.


On the right side of the screen there are four tabs:

Logged - This displays a list of users currently logged on (both in the Admin and the Frontend sections). Click a users name to go to the User Edit page for that user.

Components - Click this shortcut tab to access any of the components and component links that you wish to edit.

Popular - Click this tab to access the screen which lists in order, the most popular Content items on your site.  This includes the date the Content item was created and the number of  hits. Click the Content Item name to edit that item.

Latest Content - Click this tab to access the screen which lists the most recently added Content items.  This includes the creation date and the author. Click the Content item name to edit that item.

Menu Stats - Click this tab to access the screen which lists all menus and the number of item links in each menu.  The number of links is the total created, not the total published. Click the name of a menu to access the screen which lists the items for that menu.
